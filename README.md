''基于本地数据处理与轻量化 AI 辅助的运动助手系统''
主要面向个人日常健身与健康管理场景，解决用户运动计划不清晰、执行难以坚持以及缺乏数据反馈的问题。
系统核心逻辑采用“用户数据采集 + 智能分析 + 个性化推荐”的闭环设计：用户通过移动端记录运动数据（如步数、运动时长、消耗卡路里等），
后端基于规则引擎与简单模型对用户行为进行分析，动态生成个性化运动计划与每日目标。
